## Install

* Install the [Simple URL Chrome extension](https://chromewebstore.google.com/detail/fgfagloedegjakmfijkhpiknbdcbodhj?utm_source=blog&utm_medium=gs-link) from the Chrome Web Store
    
* Login with Google or your email via Login Link
    

## Pin Simple URL extension

To easily access the extension from all tabs, Click bookmark icon () to the right of the address bar. Then click pin next to SimpleURL.

This will add the SimpleURL button to the address bar.

# Create first Go Link (private Short URL)

Make sure you have [Simple URL Chrome extension](https://chromewebstore.google.com/detail/simpleurl-free-url-shorte/fgfagloedegjakmfijkhpiknbdcbodhj?utm_source=blog&utm_medium=post-getting-started) installed

* Enter short & easy-to-remember alias/keyword for your long URL
    
* Click 'Create Short URL' button
    

![Create Short URL with SimpleURL](https://cdn.hashnode.com/res/hashnode/image/upload/v1748292760909/1a4f50fe-c72a-4670-8980-51a0e2df6b90.png align="center")

# Try First Go Link (private short URL)

SimpleURL with works with 2 dummy domain [https://go](https://u) and [https://u](https://su)

Simply type (or simply click) `go/project` into your browser (after creating the short link in previous step)
