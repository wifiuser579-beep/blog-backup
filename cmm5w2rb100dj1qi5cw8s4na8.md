![](https://cdn.hashnode.com/res/hashnode/image/upload/v1771485782308/235757ec-ff24-40a3-b654-76004e792666.png align="center")

### **For Root Domain Setup:**

1.  Sign in to your domain provider (GoDaddy, Namecheap, Google Domains, Hostinger, etc.)
    
2.  Select your root domain
    
3.  Find the option to "Manage Name Servers"
    
4.  Replace the existing name servers with these values:
    
    *   Name server 1: `ns1.simpleurl.tech`
        
    *   Name server 2: `ns2.simpleurl.tech`
        
    *   Name server 3: `ns3.simpleurl.tech`
        
    *   Name server 4: `ns4.simpleurl.tech`
        
5.  Save the changes and wait for DNS propagation
    

**For Subdomain Setup:**

SimpleURL provides a CNAME record to add in your domain's DNS settings. Copy and paste it exactly as shown.

## **Best Practices for Using Branded Domains**

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1771485931642/4238b252-89b7-402d-a7d2-366a8bc5da5a.png align="center")

Maximize the value of your branded links with these strategies.

### **1\. Keep slugs short and memorable**

Use clear, relevant slugs that hint at the destination. `yourbrand.com/pricing` works better than `yourbrand.com/p1x2y3`.

### **2\. Use consistent naming conventions**

Develop a system for your link names. Examples: `/blog-post-title`, `/product-name`, `/campaign-name-month`.

### **3\. Leverage custom aliases for campaigns**

Create memorable campaign-specific links. `/blackfriday`, `/summer2026`, `/webinar-signup`.

### **4\. Track everything**

Check your analytics dashboard regularly. Identify which links perform best and replicate that success.

### **5\. Update destinations when needed**

SimpleURL's Pro plan lets you edit the destination URL without changing the short link. This keeps your published links permanent while maintaining flexibility.

### **6\. Match your brand voice**

Choose subdomain names that fit your brand personality. Creative brands might use `try.brand.com` while professional services might use `info.brand.com`.
